# Mealplan
A simple mealplan helper.

# Planned Feature List:
<ul>
<li>Store recipes and ingredients, with corresponding (partially optional) data fields.</li>
<li>Scale recipes based on portion numbers</li>
<li>Create a shopping list based on a collection of recipes</li> 
<li>Obtain a random collection of recipes</li>
<li>Alter recipe selection based on limits/preferences (i.e. "exclude any recipes with tomatoes" or similar) </li>
<li>Allow Users to add recipes</li>
<li>Allow Users to add recipes, while doing some checking for duplicates</li>
<li>Create nutritional overviews of recipes/mealplans</li>
<li>Organize recipes in recipebooks </li>
</ul>
